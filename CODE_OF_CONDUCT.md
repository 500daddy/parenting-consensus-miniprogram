# 行为准则 / Code of Conduct

我们希望这个项目保持友善、耐心、尊重事实，也尊重新手爸妈的不确定感。

请做到：

- 对人友善，对事认真。
- 允许不同育儿经验存在，但不要把经验当作唯一真理。
- 讨论健康内容时保持谨慎，不制造恐慌，也不淡化风险。
- 不发布骚扰、歧视、攻击、隐私泄露或商业垃圾信息。

维护者可以编辑、隐藏或关闭违反上述原则的 issue、评论和 PR。

## English

We want this project to be kind, careful, and evidence-aware.

Please:

- Be respectful to people and rigorous about ideas.
- Allow different parenting experiences without turning one experience into universal truth.
- Be careful with health-related discussions: do not create panic, and do not minimize risk.
- Do not post harassment, discrimination, personal attacks, privacy leaks, or spam.

Maintainers may edit, hide, or close issues, comments, and pull requests that violate these principles.
