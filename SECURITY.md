# 安全与健康内容边界 / Security and Health Boundary

## 报告安全问题

如果你发现安全问题，请不要公开提交 issue。请通过 GitHub Security Advisories 或私下联系维护者。

请不要在仓库、issue 或 PR 中提交：

- AppSecret、API key、访问令牌、后台登录信息。
- 真实宝宝档案、真实病历、联系方式或家庭群聊天记录。
- 未打码的用户截图。

## 健康内容边界

本项目不是医疗产品，也不提供诊断、处方、急救决策或个体化治疗建议。

内容贡献必须避免：

- 直接判断“就是某种疾病”。
- 给出药物剂量、处方或治疗方案。
- 鼓励延误就医。
- 用单个亲历经验替代通用风险提示。

内容贡献应该优先补充：

- 需要观察的信号。
- 需要记录的信息。
- 需要线下就医或急救的情况。
- 参考来源和适用边界。

## English

This project is not a medical product. It does not provide diagnosis, prescriptions, emergency decisions, or individualized treatment advice.

Do not submit secrets, real user data, baby personal information, medical records, contact details, or unredacted screenshots.

Health-related contributions should include observable signals, recordable information, risk boundaries, and reviewable sources.
