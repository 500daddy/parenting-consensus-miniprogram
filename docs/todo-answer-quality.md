# 题库答案准确性待办

## 背景

当前 mock 题库中有一批答案是从种子问题自动生成的，有些结论仍像“内容策划说明”，例如“展示三类观点”“解释不同情况”“按条件判断”。这会导致用户点进具体问题后，没有先看到直接答案。

## 下一次集中处理目标

- 把每个问题改成“先直接回答问题，再说明适用条件，最后给安全边界”的结构。
- 删除或弱化模板化尾句，尤其是非疾病类问题中不贴题的“精神、呼吸、进食饮水”统一话术。
- 同步调整首页摘要、结果页主结论、观点摘要、理由卡、权威观点摘要，避免只改一处。
- 保留“本地样例答案”和免责声明边界，不把内容写成诊断、处方或急救替代方案。
- 顺手重排问题详情页首屏，让用户一进入就能看懂“答案在哪里、先看什么、哪些是依据”。

## 详情页首屏优化待办

朋友测试反馈：第一次进入问题详情页会有点懵，不知道该看哪里、每个模块是什么意思、答案在哪里。

下次和答案校准一起处理，不先做重的新手引导，优先优化详情页本身的信息层级：

1. 首屏只突出三件事：当前问题、明确答案、风险提醒。
2. 把“先给结论”改得更像“答案卡”，标题和第一句话直接回答问题。
3. 顶部增加一个主判断标签，例如“多数情况可以”“建议观察”“需要警惕”“建议就医”。
4. 将“不同观点”“为什么这么认为”“权威来源”下沉为第二层解释，不和主答案抢视觉焦点。
5. 分享/收藏改成弱操作或右上角图标，避免和答案同级。
6. 如仍需要引导，只做一次性轻提示，例如“先看答案，再看提醒和依据”，不做全屏教程。
