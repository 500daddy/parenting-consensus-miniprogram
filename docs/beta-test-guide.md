# 内测体验说明

这是一版养娃新手村微信小程序 MVP，主要帮新手家长把常见育儿问题、不同说法和就医前要整理的信息放在一起看。

## 可以体验什么

- 搜索常见育儿问题，看多数观点、不同观点、参考来源和安全提醒。
- 收藏常看的问题，后续从底部“收藏”快速打开。
- 记录宝宝档案、就医前清单、奶量、疫苗和生长信息。
- 遇到暂时没有答案的问题，可以在“我的 - 帮我们补问题”里先记录到本机。

## 需要注意

- 当前内容仅供育儿信息参考，不能替代医生诊断、个体化用药建议或急救处理。
- 如果宝宝出现精神明显变差、呼吸异常、抽搐、尿量明显减少、持续高热、严重过敏反应、误食或卡噎等情况，请优先线下就医或联系急救。
- 当前版本还没有接后端。宝宝档案、收藏、搜索历史、工具记录和补充问题只保存在这台手机里，删除小程序或清理缓存后可能丢失。
- “帮我们补问题”目前只是本机记录，不代表已经提交到服务器。

## 怎么反馈问题

如果体验时发现内容不准确、页面显示异常、搜索不到问题或工具不好用，可以把下面信息发给邀请你体验的人：

- 问题截图
- 手机型号和系统版本
- 刚才搜索的词或所在页面
- 宝宝大概月龄
- 你原本期待看到什么

## 建议优先测试

- 搜索：发烧、咳嗽、喂奶、辅食、夜醒、疫苗、生长发育。
- 结果页：答案是否直接、参考来源是否容易理解、安全提醒是否清楚。
- 小工具：就医前清单、奶量记录、疫苗记录、生长记录。
- 我的页：宝宝档案、补充问题、反馈说明、隐私说明、免责声明。
